Gunmetal Paper Showcase

1. Introduction

This template demonstrates the key visual and structural elements of the Gunmetal paper style. It includes:

  • Dark-themed rich text containers
  • Inline sections and outline borders
  • Mathematical expressions via MathJax
  • Code highlighting with Prism.js
  • Ordered and unordered lists
  • Gradient or contrast styling (via CSS)

3. Code Showcase

Python example with syntax highlighting:

def calculate_sum(numbers):
    """Return the sum of a list of numbers."""
    total = 0
    for n in numbers:
        total += n
    return total

print(calculate_sum([1, 2, 3, 4]))
